# BringTheHash

## Introduction
BringTheHash is a simple python Hash Generator

## These are the available hashes :

- MD5
- MD4
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
- SHA3_224
- SHA3_256
- SHA3_384
- SHA3_512
- BLAKE2S
- BLAKE2B

## Usage :
```
BringMeTheHash> python3 .\bringthehash.py -v  "AnAmazingTo0l" -t md4
```
